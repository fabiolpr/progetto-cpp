# Progetto Esame C++ e Qt

## Descrizione

Questo repository contiene lo svolgimento del progetto d’esame di Programmazione C++. Il progetto è suddiviso in due parti:

- **Progetto C++**: implementazione di una coda generica
- **Progetto Qt**: sviluppo di un editor di testo minimale con interfaccia grafica
